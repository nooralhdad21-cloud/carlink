# منصة دينار وذهب — المواصفات التقنية

## 1. المفهوم والرؤية

**منصة دينار وذهب** هي منصة مالية عراقية إنسانية مصممة لحماية المواطن العراقي من الغش والاحتيال في صرف الدولار والذهب. الهدف الأساسي هو توفير معلومات دقيقة ومحدثة بشكل مباشر كل دقيقة، مع إشعارات ذكية وتطبيق موبايل، 让市民能够自信地做出财务决策.

- **الشعور العام**: موثوق، احترافي، عراقي أصيل، سهل الاستخدام
- **النمط البصري**: تصميم ليلي فاخر مع لمسات ذهبية — يذكرنا بالبورصات العالمية
- **اللغة**: عربي عراقي بسيط ومباشر

## 2. لغة التصميم

### الخطوط
- **الخط الرئيسي**: Cairo (عربي) — Google Fonts
- **الخط الاحتياطي**: system-ui, sans-serif

### نظام المسافات
- Base unit: 8px
- Border radius: 14px (cards), 8px (buttons)
- Card padding: 20px
- Section gap: 28px

### فلسفة الحركة
- انتقالات سلسة: 200-300ms ease-out
- تأثيرات hover: translateY(-2px) + glow
- تحديثات الأسعار: وميض لوني (أخضر/أحمر) للفروقات
- صوت طقطقة خفيف عند الكتابة

### العناصر البصرية
- أيقونات: Emoji native (🏧 💰 📊 ⚖️)
- Charts: Chart.js مع تدرجات ذهبية
- Gradients: radial مع نقاط ضوء ذهبية

## 4. الميزات والتفاعلات

### 4.1 شريط الأسعار المستمر (Ticker)
- **السلوك**: يتحرك أفقياً، يتوقف عند hover
- **المحتوى**: ذهب 21/24، فضة، صيرفة، موازي، رسمي، يورو، كويتي، برنت
- **التحديث**: كل 20 ثانية (محاكاة API)
- **الصوت**: صوت "ding" خفيف عند كل تحديث

### 4.2 بطاقات الأسعار
- **الرسمي**: 1,310 د.ع/دولار (ثابت)
- **الصيرفة**: 1,540 د.ع/دولار (متغير)
- **الموازي**: 1,480 د.ع/دولار (متغير)

### 4.3 الحاسبة الذكية
#### تبويب الذهب:
- اختيار العيار (10, 14, 18, 21, 22, 24)
- نوع القطعة (جديد/مستعمل/خردة)
- الوزن بالغرام
- أجور الصياغة (اختياري)
- العمولة (اختياري)
- العدد

#### تبويب الصيرفة:
- عرض معلومات منصة الصيرفة
- مقارنة الأسعار الثلاثة
- تنبيه عند تجاوز الحد اليومي (250$)

#### النتيجة:
- إجمالي القيمة
- تفاصيل تفصيلية
- زر نسخ + مشاركة
- تحليل AI

### 4.4 نظام الإشعارات
- **الإعداد**: يدخل المستخدم سعر تنبيه
- **التخزين**: localStorage
- **التنبيه**: Notification API + صوت
- **الحدود**: ذهب، صيرفة، موازي

### 4.5 قسم الأخبار
- آخر أخبار الاقتصاد العراقي
- توقعات سعر الصرف
- تحليلات السوق
- أسعار الذهب العالمية

### 4.6 تطبيق الموبايل (PWA)
- Service Worker للتخزين المؤقت
- manifest.json للتثبيت
- تصميم متجاوب بالكامل
- أيقونة مخصصة

### 4.7 مؤثرات صوتية
- **typing**: صوت طقطقة خفيف أثناء الكتابة
- **update**: رنين خفيف عند تحديث الأسعار
- **alert**: نغمة تنبيه عند الوصول للسعر المطلوب
- **success**: صوت نجاح عند إكمال العملية
